1-2 소자 설명

(왼쪽 사진 참고)

위: LCD 16x2

아래: LCD 16x2(I2C)

    - 두 개의 데이터 선을 이용하여 여러 디바이스를 연결하는 직렬 통신 프로토콜

      - LCD 사용을 위해선 라이브러리를 추가하여야 하는데,

         위의 LCD는 ‘LiquidCrydystal’, 아래는 ‘LiquidCrydystal_IC12’라는 라이브러리를 추가하여야 한다.

(왼쪽의 위 사진을 오른쪽으로 90℃ 회전)

LED- 백라이트 LED의 양극 (일반적으로 VCC에 연결)

LED- 백라이트 LED의 음극 (일반적으로 GND에 연결)

D87~D80- 데이터 핀 7~0. 핀 7, 3, 2, 1, 0는 4 비트 모드에서는 사용X

E- Enable 핀

RW- Read/Write 핀 (일반적으로 GND에 연결하여 쓰기 모드로 사용)

RS- Register Select 핀

V0- LCD 대비 조절 핀

VCC- 전원 (5V에 연결)

GND- 그라운드 (GND에 연결)
